◆ add_scheduled_event()

void add_scheduled_event ( uint32_t  event)

Function adds an event to the schedule.

add_scheduled_event() ORs the input event with the static variable event_scheduled in order to add the input event to the scheduler.

Note
Scheduler must be open before this function is called.
Parameters
[in]eventevent is unsigned integer representation of the event to be added to the schedule.

◆ get_scheduled_events()

uint32_t get_scheduled_events ( void  )

Function returns integer storing scheduled events.

get_scheduled_events() returns the unsigned int event_scheduled. Each bit in event_scheduled represents aa different event that can be added to the scheduler making 32 possible scheduled events.

Note
Scheduler must be opened with scheduler_open() before this function can be called.

◆ remove_scheduled_event()

void remove_scheduled_event ( uint32_t  event)

Function removes an event from the schedule.

removed_scheduled_event() ANDs the inverse to the input event with event_scheduled in order to remove the input event from the scheduler.

Note
Scheduler must be open before this function is called.
Parameters
[in]eventevent is unsigned integer representation of the event to be removed from the schedule.

◆ scheduler_open()

void scheduler_open ( void  )

Function opens the scheduler.

scheduler_open() initializes static variable event_scheduled to 0.

Note
This function must be called before scheduler can be used.
