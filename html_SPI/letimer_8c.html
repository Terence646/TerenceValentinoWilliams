◆ LETIMER0_IRQHandler()

void LETIMER0_IRQHandler ( void  )

Function checks if interrupt flags have been set, then adds corresponding event to scheduler.

LETIMER0_IRQHandler() sets int_flag to equal the interrupts that have been flagged and are enabled, then clears the flagged interrupts. Next, the flagged event gets added to the scheduler

Note
Each bit in int_flag corresponds to its own interrupt, uf, comp0, and comp1.

void letimer_pwm_open ( LETIMER_TypeDef *  letimer,
APP_LETIMER_PWM_TypeDef *  app_letimer_struct

Driver to open an set an LETIMER peripheral in PWM mode.

This routine is a low level driver. The application code calls this function to open one of the LETIMER peripherals for PWM operation to directly drive GPIO output pins of the device and/or create interrupts that can be used as a system "heart beat" or by a scheduler to determine whether any system functions need to be serviced.

Note
This function is normally called once to initialize the peripheral and the function letimer_start() is called to turn-on or turn-off the LETIMER PWM operation.
Parameters
[in]letimerPointer to the base peripheral address of the LETIMER peripheral being opened
[in]app_letimer_structIs the STRUCT that the calling routine will use to set the parameters for PWM operation

◆ letimer_start()

void letimer_start ( LETIMER_TypeDef *  letimer,
bool  enable 
)

Function to enable/turn-on or disable/turn-off the LETIMER specified.

letimer_start uses the lower level API interface of the EM libraries to directly interface to the LETIMER peripheral to turn-on or off its counter. This function blocks EM4 if turning on LETIMER and unblocks sleep mode when turning off LETIMER.

Note
This function should only be called to enable/turn-on the LETIMER once the LETIMER peripheral has been completely configured via its open driver
Parameters
[in]letimerPointer to the base peripheral address of the LETIMER peripheral being opened
[in]enableVariable to turn-on the LETIMER if boolean value = true and turn-off the LETIMER if the boolean value = false
