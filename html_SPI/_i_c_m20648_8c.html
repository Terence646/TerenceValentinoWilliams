◆ ICM20648_Config()

void ICM20648_Config ( void  )

ICM20648_Config function is used as the Test Driven Development for this application.

Using a series of "Write - Read - Assert" in order to verify that the Sensor is configured properly.

Note
Must use a timer delay of 22us in order to allow time to process between each read and write.
Parameters
[in]NONE

◆ ICM20648_open()

void ICM20648_open ( void  )

ICM20648_open initializes the local SPI setup structure.

This function opens the driver by initializing various parameters such as the TX and RX usart route, as well as the RX and TX pin enables.

Note
Auto CS enable is set to false due to the driver manually enabling and disabling the chip select.
Parameters
[in]NONE
