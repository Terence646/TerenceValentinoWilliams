void cmu_open ( void  )

Within the cmu.c module, this is responsible for enabling all oscillators and routing the clock tree for the application.

In doing so, we use these oscillators through the clock tree for intended specifications to blink our green LED else where in the project.

We disable two oscillators and enable our intended one.

Note
No requirement to enable the ULFRCO oscillator. With this we have set our clock tree which can be found in the reference manual.
